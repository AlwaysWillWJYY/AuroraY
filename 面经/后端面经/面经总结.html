    4.SpringBoot里面用了哪些starter?

    Spring Boot 提供了大量的 Starter（启动器），它们为常见的开发任务提供了预先配置好的依赖项和设置，让开发人员能够更快地构建应用程序。以下是 Spring Boot 常用的 Starter：

    1. spring-boot-starter：核心 Starter，包含了 Spring Boot 应用所需的所有基础依赖。
    2. spring-boot-starter-web：用于构建 Web 应用程序，包含了 Spring MVC、Tomcat 和 Spring Websocket。
    3. spring-boot-starter-data-jpa：用于构建基于 JPA 的数据访问层应用程序，包含了 Hibernate、Spring Data JPA 和 HikariCP 连接池。
    4. spring-boot-starter-data-redis：用于构建 Redis 数据访问层应用程序，包含了 Redisson、Spring Data Redis 和 Jedis 连接池。
    5. spring-boot-starter-test：用于构建测试应用程序，包含了 JUnit、Spring Test、Mockito 和 Hamcrest。
    6. spring-boot-starter-actuator：用于监控和管理应用程序，包含了许多端点（Endpoints），可以查看应用程序的健康状态、运行指标等信息。
    7. spring-boot-starter-security：用于构建安全性应用程序，包含了 Spring Security、JWT 和 OAuth2。
    8. spring-boot-starter-logging：用于构建日志应用程序，包含了 Logback 和 Log4j2。

    以上是一些常见的 Spring Boot Starter，开发人员可以根据实际需求选择相应的 Starter 进行引入。
